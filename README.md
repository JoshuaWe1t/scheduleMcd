# Навигация по проекту

## Backend

```
miniProjectSchedule/
│
├── backend/
│   ├── app/                      # Основной пакет приложения
│   │   ├── __init__.py           # Инициализация приложения
│   │   ├── main.py               # Точка входа (если нужно)
│   │   ├── routes.py             # Маршруты / эндпоинты
│   │   ├── models.py             # Модели данных (ORM)
│   │   ├── schemas.py            # Схемы валидации (Pydantic и т.п.)
│   │   ├── services/             # Логика приложения (сервисы)
│   │   └── utils.py              # Вспомогательные функции
│   │
│   ├── tests/                    # Тесты backend
│   │   └── test_main.py
│   │
│   ├── requirements.txt         # Зависимости Python
│   ├── Dockerfile               # Если планируете контейнеризацию
│   ├── .env                     # Файл с переменными окружения (в .gitignore)
│   ├── README.md                # Документация по backend
│   └── run.py                   # Скрипт запуска приложения
│
├── src/                # Исходные файлы проекта
│   ├── assets/         # Медиафайлы (изображения, шрифты и т.д.)
│   ├── css/            # Стили CSS
│   ├── js/             # JavaScript файлы
│   │   ├── modules/    # Модули JavaScript (если есть)
│   │   └── app.js      # Главный файл JavaScript
│   └── index.html      # Главный HTML файл
│
├── dist/               # Собранные файлы для продакшена
│
└── README.md           # Документация проекта
│
└── scheme              # Папка со схемами работы приложения
```
